AutoList

Introduction

AutoList is a program that extracts information from AutoCAD List command output. Currently this can be used with text, mtext, lines, lwpolylines and hatches. This tool can be used to calculate the total length/area of lines and hatches or download all of the lengths/areas of many different hatchs/lines/arcs and so on.

How to

Within AutoCAD first enter the LIST command, then select the objects that you would like to see information about. Then just paste the whole output of the LIST command into the text bow below.

More than one object

When you would like to download a CSV of these objects select the objects in the following order (TEXT, LINE/LWPOLYLINE/HATCH/ARC, TEXT, LINE/LWPOLYLINE/HATCH/ARC, ...). The text object will be used as the object's ID. The other objects are not required. Note that if selecting more than one LINE/LWPOLYLINE between TEXTs the lengths will be added together to give a total length. The same goes for HATCHs but for their areas.
